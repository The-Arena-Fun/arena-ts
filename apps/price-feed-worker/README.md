## Project setup

1. Install dependencies:
```bash
$ yarn
```

2, Setup .env
```.env
RPC_URL= # Solana RPC URL
SUPABASE_URL= # Supabase URL
SUPABASE_KEY= # Supabase access key
SUPABASE_FEED_TABLE_NAME= # Supabase price feed table name
```

3. Build
```bash
$ yarn build
```

4. Start
```bash
$ yarn start
```
